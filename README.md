# Snake Game

Snake game built with **Vite** + **Svelte**, **TypeScript**, and **HTML5 Canvas**. Play Snake Game directly in your browser with keyboard controls or touch-friendly mobile buttons.

## Features
- Snake gameplay
- Keyboard controls (Arrow keys)
- Mobile-friendly on-screen controls
- Score tracking
- Game over detection
- Smooth animations with canvas rendering
- Moving Snake
- Fully responsive design

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yarosprojects/snake-yarosprojects.git
cd tetris
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser at http://localhost:5173 (or the port shown in terminal).
